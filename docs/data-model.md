# PocketBase Collections 草案

当前不实现任何 PocketBase collection，本文件仅记录未来可能的数据方向。

## 可能的 collections

- `users`
  - 基础账号信息
  - 头像、昵称、权限标记
- `profile_sections`
  - 首页展示的模块配置
  - 排序、标题、描述、显示状态
- `moments`
  - “我还在”或动态类内容
  - 文案、封面、时间、状态
- `albums`
  - 相册信息
  - 标题、描述、可见性
- `ledger_entries`
  - 记账条目
  - 金额、分类、日期、备注

当前阶段不创建、不迁移、不接 SDK。
