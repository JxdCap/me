# UI & UX 交互规范 (Design Spec)

本项目定位为个人“数字杂志 (Zine)”，强调极致的克制、物理真实感与沉浸式体验。

## 1. 核心视觉原则
*   **极简主义 (Minimalism)**：移除所有非必要的装饰。页面平时静如止水，仅通过留白和排版传达力量。
*   **物理手感 (Physicality)**：所有的位移、回弹、拖拽都遵循非线性的物理规律（Spring Physics）。
*   **环境感知 (Ambient Intelligence)**：背景带有交互式点阵网格，随鼠标/手指产生视差位移；支持深夜模式持久化。

## 2. 布局架构：单屏视窗 (Single-Screen System)
*   **容器限制**：外层容器强制锁定 `100dvh`，屏蔽原生滚动。
*   **空间分配**：
    *   **顶部 (Topbar)**：极简功能岛。
    *   **中部 (Hero)**：品牌身份区，占据屏幕重心（Optical Center）。
    *   **底部 (Memos)**：动态堆叠区，占据剩余空间。
*   **PC 适配**：采用“双栏社论布局”。左侧展示身份，右侧展示动态，形成平衡的宽屏艺术感。

## 3. 核心交互组件

### 3.1 技能标签 (Skill Tags)
*   **形态**：高对比度药丸按钮。
*   **交互**：点击标签触发“抽屉式”展开。展开时其他标签自动降噪（40% 透明度 + 灰度），强化视觉焦点。

### 3.2 正在录入 (Still Alive Stack)
*   **堆叠逻辑**：模拟 iOS 锁屏通知，采用 `translate3d` 在 Z 轴上构建真实纵深。
*   **物理引擎**：支持 1:1 跟手拖拽。
    *   **上滑 (>60px)**：飞出当前卡片，进入下一条。
    *   **松手 (<60px)**：弹簧超调回弹 (Overshoot Snap-back)。
*   **无限循环**：采用 4 层缓冲区渲染，确保切换瞬间视觉不坍塌。

### 3.3 沉浸阅读器 (Zine Reader)
*   **入场**：从卡片位置执行“物理扩张”动效，背景同步“视窗推后 (Pushed-back)”。
*   **内容**：
    *   **社论网格**：根据图片数量（1-9）自动生成非对称、高完成度的杂志排版。
    *   **渐进加载**：图片预留比例占位，通过骨架屏扫光到模糊淡入（Progressive Loading）平滑过渡。
    *   **连续流**：支持向下无限滚动回溯历史记录。

### 3.4 交互导航 (The Index)
*   **形态**：全屏深度模糊蒙层。
*   **动效**：链接项带有 Staggered Delay 从下方依次升起，图标随悬停产生视差。
